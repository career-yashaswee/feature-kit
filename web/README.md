# Feature Kit

A collection of reusable web features and components.

## Features

- **Auto Save Form** - Automatically saves form data with debouncing
- **Export Button** - Export data to CSV or JSON format with toast notifications
- **Network Status Listener** - Monitor network connectivity with toast notifications
- **Optimistic Action Button** - Perform actions with optimistic UI updates
- **Refresh Button** - Invalidate React Query cache with loading states
- **Scroll to Top** - Smooth animated scroll-to-top button
- Stateful Button
- Table of Contents
- In Game Currency Details
- Leaderboard
